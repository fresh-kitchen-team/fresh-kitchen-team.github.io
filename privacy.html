개인정보 처리방침

FreshKitchen

시행일: 2026년 6월 1일
최종 개정일: 2026년 6월 1일

FreshKitchen(이하 '회사' 또는 '서비스')은 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 등 관련 법령을 준수하기 위하여 노력하고 있습니다. 본 처리방침은 회사가 제공하는 FreshKitchen 모바일 애플리케이션 서비스 이용과 관련하여, 이용자의 개인정보가 어떻게 수집·이용·보관·파기되는지 안내합니다.

1. 수집하는 개인정보 항목

회사는 서비스 제공을 위해 다음과 같은 개인정보를 수집합니다.

구분수집 항목
계정 정보Google / Kakao 소셜 로그인 식별자(ID), 이메일, 닉네임, 프로필 이미지 URL
서비스 이용 정보식재료 등록 내역, 유통기한 정보, 레시피 즐겨찾기, 소비 이력
AI 챗봇 이용 정보AI 채팅 입력 내용 및 응답 기록
이미지 정보식재료·영수증 스캔 이미지 (인식 처리 목적)
기기 정보FCM 푸시 토큰, OS 버전, 앱 버전

2. 개인정보 수집 및 이용 목적

  • 앱 기능 제공: 식재료 관리, 레시피 추천, AI 챗봇, 스캔 기능 제공
  • 계정 관리: 회원 식별, 로그인 유지, 회원 탈퇴 처리
  • 푸시 알림: 유통기한 임박 알림, 서비스 공지 발송
  • 서비스 분석 및 개선: 이용 패턴 분석, 기능 개선, 오류 대응

3. 개인정보 보관 및 파기

  • 회원 탈퇴 요청 시 회사는 보유한 개인정보를 지체 없이 파기합니다.
  • 다만, 일시적 오용·악용 방지 및 분쟁 대응을 위해 일정 기간(최대 30일) 소프트 삭제 상태로 보관 후 영구 삭제될 수 있습니다.
  • 관련 법령에 따라 보관이 필요한 경우 해당 법령에서 정한 기간 동안 보관합니다.

4. 개인정보의 제3자 제공

회사는 원칙적으로 이용자의 개인정보를 외부에 제공하지 않습니다. 다만, 다음의 경우에 한해 제휴 서비스가 동작합니다.

제휴사제공 형태목적
쿠팡 파트너스 개인정보 미제공 (쿠팡 상품 검색 API 호출 시 검색 키워드만 전달) 식재료 구매 추천 상품 표시
Google Gemini API 개인 식별이 불가능한 형태로 식재료·레시피·채팅 텍스트 전달 AI 레시피 추천 및 챗봇 응답 생성

※ 외부 API 연동 시 이용자의 개인 식별정보(이름, 이메일 등)는 일절 전달되지 않으며, 서비스 제공에 필요한 최소한의 텍스트만 전달됩니다.

5. 개인정보 처리 위탁

회사는 서비스 제공에 필요한 일부 업무를 외부 전문 업체에 위탁할 수 있으며, 위탁 시 관련 법령에 따라 안전한 관리를 위한 조치를 마련합니다. 주요 위탁 처리 항목은 클라우드 인프라, 푸시 알림 발송(Firebase Cloud Messaging), AI 분석 API 등입니다.

6. 이용자의 권리

  • 이용자는 언제든지 본인의 개인정보를 조회·수정·삭제·처리정지 요청할 수 있습니다.
  • 앱 내 설정 메뉴에서 회원 탈퇴 시 모든 계정 데이터가 삭제됩니다.
  • 기타 요청은 아래 연락처를 통해 처리할 수 있습니다.
